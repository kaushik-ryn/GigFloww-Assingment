📌 Overviee

This project is a modern and responsive Login Page built as part of the GigFloww assignment.
The focus was on delivering a clean UI, smooth UX, and user-friendly design using frontend technologies.

🎯 Features
✨ Clean and modern UI design
📱 Fully responsive (mobile + desktop)
🔐 User-friendly login form
🎨 Smooth styling with good color balance
⚡ Fast and lightweight implementation
🧭 Simple navigation and intuitive layout


🛠️ Tech Stack
HTML5 – Structure
CSS3 – Styling & responsiveness
JavaScript – Basic interactivity


📂 Project Structure
GigFloww-Assignment/
│── index.html     # Main login page
│── style.css      # Styling file
│── script.js      # JavaScript logic
│── img/           # Images & assets
│── README.md      # Project documentation


💡 UI/UX Highlights
Minimalistic design for better focus
Proper spacing and alignment
Input fields designed for clarity and ease of use
Consistent typography and color scheme
Responsive layout for all screen sizes
